# referee_reports/ — output

Generated referee reports land here, named:

```
referee_report_<sanitized-paper-name>_<YYYY-MM-DD>.md
```

Each report follows `templates/referee-report.md`: a summary assessment with an overall
recommendation, strengths, major/minor concerns, referee objections, and a 1–5 ratings table.

`referee_reports/.work/` holds the intermediate **paper brief** and is gitignored — scratch space, not output.
